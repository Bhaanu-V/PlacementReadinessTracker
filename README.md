# Placement Readiness Tracker

## Overview

Placement Readiness Tracker is a Java console-based application that helps students monitor their placement preparation.

The application allows users to:
- Manage student profile
- Track completed DSA topics
- Track solved problems
- Store aptitude scores
- Maintain resume checklist
- Track target companies
- Generate placement readiness report
- Save progress to a file
